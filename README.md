A MATLAB code that can generate discrete fault networks, as described in the Im (2023) article. Please note that it should not be used for commercial purposes. Reference:
---
The model does not account for dynamic stress transfer mediated by seismic wave propagation. Fault friction is governed by the rate-and-state friction law, with velocity-weakening friction applied on all faults <br>
In the model, shear stress increases uniformly over time to represent far-field tectonic loading. This loading method plays an important role: if there is no interaction among faults, each fault will undergo complete rupture at approximately fixed recurrence intervals <br>
>> Im, K., & Avouac, J.-P. (2023). Cascading foreshocks, aftershocks and earthquake swarms in a discrete fault network. Geophysical Journal International, 235(1), 831–852. https://doi.org/10.1093/gji/ggad278
